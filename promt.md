Saya memiliki aplikasi Laravel Jetstream (gunakan Tailwind CSS, komponen Blade/Livewire).

Saat ini tampilannya masih default Jetstream (minimalis dan basic). Saya ingin melakukan redesign UI/UX tanpa mengubah struktur backend.

Tujuan redesign:

Tetap mempertahankan kesederhanaan Jetstream, tetapi dengan tampilan yang lebih modern, profesional, dan polished
Meningkatkan visual hierarchy, konsistensi komponen, dan user experience

Kriteria desain:

Gunakan Tailwind CSS (tidak boleh mengganti ke framework lain)
Terapkan prinsip modern UI:
spacing lebih lega
typography lebih jelas (heading, subheading, body)
penggunaan shadow, border radius, dan layering
Komponen yang perlu ditingkatkan:
navbar (lebih clean, responsif, modern)
form (input, label, error state, focus state)
button (primary, secondary, danger, dengan hover & active state)
card / container layout
Tambahkan UX improvements:
loading state (spinner/skeleton)
feedback (success/error message lebih jelas)
hover & transition halus
Gunakan pendekatan reusable component (Blade component jika memungkinkan)

Output yang saya inginkan:

Contoh redesign untuk halaman dashboard dan auth (login/register)
Kode Tailwind yang clean dan reusable
Penjelasan perubahan (apa yang di-upgrade dan kenapa)

Jangan hanya mengganti warna. Fokus pada peningkatan kualitas desain secara menyeluruh.

sesuaikan juga nama aplikasinya sesuai dengan sistem saya
